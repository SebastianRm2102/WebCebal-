El reciclaje de papel es el proceso mediante el cual se recolecta, se procesa y se reutiliza papel desechado para fabricar nuevos productos de papel. Este proceso contribuye a la conservación de los recursos naturales, la reducción de la contaminación y la disminución de la cantidad de residuos que terminan en los vertederos.

  • Recolección: El primer paso es la recolección del papel usado. Esto puede incluir periódicos, revistas, papel de oficina, cajas de cartón, etc. La recolección se realiza mediante programas de reciclaje doméstico, comercial e industrial.
  • Clasificación y Separación: Una vez recolectado, el papel se lleva a una planta de reciclaje donde se clasifica y separa según el tipo y la calidad del papel. Esto es importante porque diferentes tipos de papel tienen diferentes propiedades y usos.
  • Triturado: El papel clasificado se tritura en pedazos pequeños para facilitar el proceso de reciclaje.
  • Desintegración y Pulpeado: El papel triturado se mezcla con agua y se agita en grandes tanques para formar una pulpa. Este proceso descompone las fibras del papel y elimina cualquier material no deseado, como tinta, adhesivos o grapas.
  • Descontaminación: La pulpa se limpia y se filtra para eliminar contaminantes. Esto puede incluir el uso de productos químicos para eliminar la tinta (proceso conocido como desentintado) y otros residuos.
  • Refinamiento y Blanqueo: La pulpa limpia se refina y, si es necesario, se blanquea para mejorar su color y calidad. El blanqueo puede requerir productos químicos adicionales, aunque algunos procesos de reciclaje se esfuerzan por ser libres de cloro para minimizar el impacto ambiental.
  • Formación de Nuevas Hojas: La pulpa refinada se extiende sobre pantallas o mallas finas para formar nuevas hojas de papel. Estas hojas se prensan para eliminar el exceso de agua y se secan para obtener el producto final.
  • Corte y Enrollado: El papel seco se corta y se enrolla en grandes bobinas o se transforma en otros productos, como cuadernos, papel higiénico, cajas de cartón, etc.
  • Distribución: Finalmente, el papel reciclado se distribuye y se vende para su uso en diversas aplicaciones.

  • Periódicos y revistas
  • Cartón
  • Papel de cuaderno
  • Cajas de embalaje
  • Papel de impresora y fotocopiadora
  • Folletos y catálogos

Conservación de Bosques

El reciclaje de papel reduce la necesidad de talar árboles, protegiendo los bosques y los hábitats de la fauna.

Ahorro de Energía

La producción de papel reciclado consume menos energía que la producción de papel nuevo a partir de madera virgen.
